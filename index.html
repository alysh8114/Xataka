Atajos de teclado para pestañas y ventanas

A la hora de operar con nuestro navegador es importante tener el contror de las pestañas. Esto no solo implica utilizar los clasicos atajos para abir ventanas y pestañas normales o en modo incognito, sino tambien el conseguir facilitar la navegacion entre ellas sin necesidad de tener que tocar el raton y perder tiempo mientras escribimos.

FUNCIÓN ATAJO WINDOWS Y LINUX ATAJO MACOS
ABRE UNA NUEVA VENTANA Ctrl + N ⌘ + N
ABRE UNA NUEVA VENTANA EN MODO INCÓGNITO Ctrl + Mayúscula + N ⌘ + Mayúscula + N
ABRE UNA NUEVA VENTANA Y VE A ELLA AUTOMÁTICAMENTE Control + T ⌘ + T
REABRE UNA PESTAÑA RECIÉN CERRADA E IR A ELLA Ctrl + Mayúscula + T ⌘ + Mayúscula + T
VE A PESTAÑA SIGUIENTE Ctrl + Tab o Ctrl + AvPág ⌘ + Opción + Der
